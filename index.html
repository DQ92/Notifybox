Simple Notifybox JavaScript & CSS only

It's simple to use, auto-resizable width and without any dependencies (even jQuery)


HOW TO USE

Use bower:

$ bower install Notifybox

or download source files

GITHUB

Don't forger to add files to your project

    <script src="dist/notifybox.js"></script>
    <link rel="stylesheet" type="text/css" href="dist/notifybox.css">



Initialize Notifybox

var notifybox = new Notifybox();

Try it!

Now, simply call function to show success notification. If you want to close just tap on icon.

notifybox.showSuccess(title, msg, time);


notifybox.showError(title, msg, time);


notifybox.showInfo(title, msg, time);


notifybox.showWarning(title, msg, time);



You can pass own title, message and duration time (default 2000 [ms])
